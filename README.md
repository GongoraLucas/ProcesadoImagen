# Comparacion entre procesamiento de imagenes en paralelo y  en secuencial 

## Uso
1. Realiza un:  git clone https://github.com/GongoraLucas/ProcesadoImagen
2. Crear en la raiz las carpetas: imagenes, imagenes_grises_concurrente e imagenes_grises_secuencial
3. En Visual Studio Code, aplasta las teclas: ctrl + shift + c  -> para abrir una terminal en la ruta donde se encuentra el proyecto.
4. Compilalo con: javac main/Main.java
5. Ejecutalo con: java main.Main
